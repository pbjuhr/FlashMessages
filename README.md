FlashMessages
=============
Php class for generating and displaying flash messages.

How to use
-------------

###1. Download

The easiest way is install using composer.
To do so, add the following to your composer.json: 

```javascript
"require": {
    "pbjuhr/flashmessages": "dev-master"
},
```
